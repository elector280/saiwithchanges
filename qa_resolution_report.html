4. Navigation Accessibility Fix
File: resources/views/frontend/layouts/header.blade.php

The desktop "More Ways To Help" dropdown trigger was implemented as a href="#", causing the page to scroll to the top when tapped on touchscreen devices and creating a semantic HTML accessibility issue. It was replaced with a button type="button" element, preserving the same visual appearance while eliminating the unintended navigation behavior and improving semantic HTML structure.
